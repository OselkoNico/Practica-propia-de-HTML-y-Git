Cosas que he aprendido con HTML y Git

¿Qué es HTML?

HTML son los cimientos de las páginas web.

¿Qué es Git?

Git es un controlador de versiones, que se utiliza para poder almacenar todos los cambios que ha ido teniendo el proyecto en el que se va trabajando. También se usa para poder trabajar a la vez en los proyectos con su sistema de ramas y así, poder hacerlo simultáneamente.

Diferentes tipos de listas de HTML

La primera lista que vamos a ver, es un tipo de lista desordenada. Lo que quiere decir con desordenada es que no tiene numeración alguna, si no que cada punto en la lista se compone de un cirlulillo. Se vería de la siguiente forma:

  • Aquí tienes un ejemplo de ello.
  • Complementando el ejemplo.

Otra forma sería una lista ordenada. Esta en vez de circulillos, estaría ordenada numericamente. Se vería de la siguiente forma:

  1. Aquí teneis un ejemplo de ello.
  2. Complementando el ejemplo.

Existes muchos más tipos de listas, por ejemplo podría ser una lista de links. Hay muchas formas de listas de links, una puede ser para moverse a traves de la página. Se veria de la siguiente forma:

  • Aquí podréis ir al título de la página.
